# Campaign Rules

Homebrew rules for the Blackmoor campaign. These override or extend standard 5e rules.

## Spell Risk

When a player casts a spell from a scroll or uses a spell card, they must roll a **Risk check**. If the roll falls within the Risk %, the spell is lost — the scroll crumbles or the spell fades from their memory.

### Default Risk by Level

| Spell Level | Risk % |
|-------------|--------|
| 1           | 10%    |
| 2           | 20%    |
| 3           | 30%    |
| 4           | 40%    |
| 5           | 50%    |
| 6+          | Set by the DM (no default) |

- **Levels 1–5**: If the DM doesn't set a Risk %, the system uses `level × 10%` automatically.
- **Level 6+**: The DM must explicitly set the Risk % when creating the item. High-level magic is too variable for a simple formula.
- **Cantrips (level 0)**: No risk — cantrips are always safe.
- The DM can override the default for any spell at any level.
